# NYC Subway Ridership Analysis

## Overview
This project analyzes New York City subway ridership trends using real-world data from NYC Open Data. The goal is to explore how ridership changed over time, focusing on the impact of COVID-19, and to visualize both overall usage and recovery trends.

## Project Goals
- Track daily subway ridership trends over time  
- Visualize recovery relative to pre-pandemic baselines  
- Highlight trends using rolling averages to smooth out daily fluctuations

## Tools Used
- Python  
- Pandas for data cleaning and analysis  
- Matplotlib for data visualization  
- Jupyter Notebook for interactive exploration

## Key Insights
- Subway ridership dropped sharply in March 2020 due to COVID-19.  
- Recovery has been gradual, with fluctuations influenced by seasonality and external events.  
- Rolling averages provide a clearer view of long-term trends, reducing daily volatility.
